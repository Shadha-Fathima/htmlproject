What is HyperText Markup Language?

HTML is the computer language that facilitates website creation. The language has code words and syntax just like any other language. It is the basic scripting language used by web browsers to render pages on the world wide web.

HTML Tags

The tags are what separate normal text from HTML code. Tags are the words between what are known as angle brackets, which allow graphics, images, and tables to appear on the web page. There are two types of tags. They are single ended tags and double ended tags. These are the basic tags in HTML:

  • HTML Headings

Html headings are titles or subtitles that you want to display on a webpage.

  • HTML Paragraphs

  • HTML Emphasized tag

This tag is used to define emphasized text. The content inside is typically displayed in italic.

  • HTML Break tag

This tag inserts a single line break which has no end tag.

  • HTML Strong tag

The strong tag is used to define text with strong importance. It is typically displayed in bold.

  • HTML ul tag

The tag defines an unordered (bulleted) list. It is used with li tag.

  • HTML ol tag

The tag defines an ordered list which can be numerical or alphabetical.

  • HTML div tag

This tag defines a division or a section in an HTML document.

  • HTML img tag

This tag is used to embed an image in an HTML page.
